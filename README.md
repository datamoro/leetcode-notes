# leetcode-notes

Starter repo para registrar estudos de **SQL** e **DSA** (LeetCode-style).

## Como usar
1. **Encontre o problema no LeetCode** (barra de busca) pelo **título exato** (veja `progress.csv` ou os READMEs por padrão).
2. Resolva no site. Depois, copie sua solução para este repo:
   - SQL → `sql/<pattern>/<problem-slug>/solution.sql`
   - DSA (Python) → `dsa/<pattern>/<problem-slug>/solution.py`
3. Documente rapidamente em `README.md` do problema: abordagem, casos de teste e Big-O.
4. Atualize `progress.csv` com *status*, *tempo* e *revisões* (D+1/D+7).

## Dicas
- Prefira **padrões** (hashing, two pointers, sliding window, binary search, heaps, intervals, BFS/DFS).
- Para SQL, foque em joins, agregações condicionais, **window functions** e anti-joins.
- Commit curto e descritivo: `feat(sql/window_functions): nth_highest_salary`.
